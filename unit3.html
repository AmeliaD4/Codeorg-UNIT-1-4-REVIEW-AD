What is the difference between a sequential program and an event-driven program?
A sequential program only goes in order from top to bottom regardless of where everything is placed, while an event-driven program has elements only work when their haptic is heard, and they can work indefinitely, regardless of what order the code is in. Events work separately from the sequential code.
